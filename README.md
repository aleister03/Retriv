# Retriv
A community driven web-based project which helps user to report and find lost items, buy-sell old/new products, borrow/rent products between users. 
It is being made for CSE470 project.
Contributors: [Aorko Shikder](https://aleister03.github.io/) | [Shruti Mandal Proma](https://github.com/shrutipromamandal-prog) | [Fayrooz Tahmida](https://github.com/Fayrooz0) | [Samia Ali](https://github.com/samiaali2016)

## Authentication

### Admin Login
- Email: `admin@system.com`
- Password: `admin123`
- Access the admin dashboard after successful login

### Student Registration
- Create a student account with the following information:
  - Full Name
  - Email (must end with @g.bracu.ac.bd)
  - Student ID
  - Semester
  - Department
  - Phone Number
  - Password (minimum 6 characters)
- After registration, student is automatically logged in and redirected to the student dashboard

### Student Login
- Login with your registered email and password
- Access the student dashboard after successful login
